# Baseline Examples

Each executable under `src/apps/` is a project-local reconstruction of a baseline example.

- `ex01_nanovdb_hello`: simple level-set sampling and gradients.
- `ex02_hppfcl_distance`: reference distance/collision query through the interface layer.
- `ex03_dual_sdf_gap`: symmetric dual-SDF gap and contact frame extraction.
- `ex04_single_step_contact`: single-step contact problem assembly and solve.
- `ex05_compare_backends`: compare reference geometry and dual-SDF results.
- `ex06_regression_smoke`: lightweight end-to-end smoke executable.
